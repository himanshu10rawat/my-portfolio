Each One Teach One

ICAI Each one teach one website portal

The ICAI Each One Teaches One website portal features a streamlined front-end interface for managing mentorship programs. Users begin by entering a mobile number to receive a One-Time Password (OTP) for login, after which they can create and edit their account details and upload a profile photo. Once the account is set up, users can add and manage student profiles and schedule mentorship sessions, including specifying meeting dates, modes, and activities. Additionally, users can download mentorship details in PDF, CSV, and Excel formats. Administrators have access to a comprehensive dashboard that allows them to monitor and manage all teacher and student records efficiently.
